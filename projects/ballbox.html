Ball-to-box, a Python game

This is a small game project I made in my spare time with the aim of learning the programming language Python. It is a simple puzzle game with the goal of bouncing the red ball to the green goal block. The project uses pygame for drawing primitive shapes, fonts and the viewport. A ported version of the physics enginge Box2D was used for collision detection and interaction between the objects in the game and with the mouse cursor. The project was developed in Ubuntu.
